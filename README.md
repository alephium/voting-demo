# My dApp

## Install

```
npm install
```

## Start the app

Compile the TypeScript files into JavaScript:

```
npm run start
```

## Start a devnet

```
npm run start-devnet // this will start a devnet for smart contract tests
```

## Stop/restart devnet

```
npm run stop-devnet
npm run restart-devnet
```

## Testing

```
npm test
```
